# Brand

The source code is open source. The Cumulus and Relay names, marks, domains,
and visual identity are protected by the trademark policy in `TRADEMARKS.md`.

Forks can keep attribution to upstream Cumulus Relay, but public hosted forks
should use their own product name and visual identity.

## Voice

Cumulus Relay should read as:

- direct,
- technical,
- calm,
- trustworthy,
- operator-friendly.

Avoid hype. Explain capabilities, tradeoffs, and operational requirements
plainly.

## Visual Direction

The product UI uses a restrained developer-tool style:

- high readability,
- dense but organized information,
- strong whitespace,
- simple borders,
- minimal decoration,
- accessible contrast,
- no unnecessary animation.

Public forks should replace Cumulus-specific marks, domains, and brand assets.
