## Replacing a track

If you swap out a track:

1. Drop the new `.mp3` into the same vibe directory.
2. Either delete the old file (so the alphabetical default picks the new one) or pass `--music <path>` explicitly to the skill.
3. Update the row above with title, artist, source URL, and download date — including the license, if not Pixabay.
4. If the new track is **not** under Pixabay Content License, add a separate license file alongside it documenting the actual terms.
